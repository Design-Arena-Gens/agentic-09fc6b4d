This project is a Next.js experience site that presents a cinematic product walkthrough for **NoSkip**, an AI-powered habit tracker focused on camera verification, adaptive coaching, and tangible rewards.

### Tech Stack
- Next.js App Router with TypeScript
- Tailwind utility classes alongside custom CSS modules for motion design
- Responsive layout tuned for Vercel deployment

### Local Development
```bash
npm install
npm run dev
```
Open http://localhost:3000 to preview the interactive promo experience.

### Production Build
```bash
npm run build
npm run start
```

### Deployment
The project is configured for one-command Vercel deployment:
```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-09fc6b4d
```
